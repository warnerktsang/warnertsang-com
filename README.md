# warnertsang.com

Static Astro project index for Warner Tsang.

## Project Structure

```text
/
├── public/
├── src/
│   ├── content/
│   │   └── projects/
│   ├── content.config.ts
│   ├── layouts/
│   │   └── Layout.astro
│   └── pages/
│       └── index.astro
├── AGENTS.md
└── package.json
```

Project cards are sourced from markdown frontmatter in `src/content/projects/`.

## Commands

```sh
npm install
npm run dev
npm run build
npm run preview
```
